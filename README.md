# Calculator Web App

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.  
This project demonstrates basic DOM manipulation, event handling, and UI styling.

---

## Live Demo
https://seeram17.github.io/Calculator/

---

## Features
- Basic arithmetic operations (+, −, ×, ÷)
- Horizontally scrolling display for long calculations
- Right-aligned calculator display
- Persistent calculation using localStorage
- Smooth hover and click animations
- Clean dark-mode inspired UI

---

##  Technologies Used
- HTML5
- CSS3 (Grid, Transitions)
- JavaScript (DOM Manipulation, Events, localStorage)

---

##  What I Learned
- Handling user input using JavaScript
- Updating the DOM dynamically
- Using localStorage to persist data
- Creating layouts with CSS Grid
- Improving user experience with animations and scrolling behavior
